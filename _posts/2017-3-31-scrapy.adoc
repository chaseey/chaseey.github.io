# 插入数据库出错
向mysql中插入数据时出现异常

	_mysql_exceptions.OperationalError: (1366, "Incorrect string value: '\xE4\xB8 ......' for column 'pageSnapshot' at row 1")


需要将表字符集修改为 `utf8_general_ci`

	mysql> set names utf8;	# fix temporary

```
# my.cnf for mysql5.5
[mysqld]
character-set-server = utf8
```
